# 输出

长期以来对于读书都是一种酒肉穿肠过，佛祖留心中的态度，我不想也懒得写一些文字来总结自己对某本书的阅读体验和相关感想。以此类推对电影电视剧和大部分的人事物都是这个态度。我总觉得隐式地内化比借由输出内容而产生的内化高级一些似的。

但是今天整理Kindle书架的时候我发现有一本书明明被标记了已读完，但我却对它的内容基本毫无印象（一部分原因可能是我在大一的时候读的这本书，三年时间足以冲淡很多东西了，何况一本我强制自己读的、我并不太感兴趣的领域的书的内容）。犹豫半天我还是把它放进“已读完”类别了。

整理完Kindle我想起前几天看到的一条评论，“长期坚持输入对个人固然是好事，然而，培养习惯容易，这种成长就像一场不用考试的学习，只要自己认可就行。但是打造技能很难，因为技能好坏全由外界评定”。

确实，而且只有输入的话，接受的内容的价值会被迅速稀释，甚至都不是稀释，是蒸发了。因为大部分我需要学习的东西，在自然情况下，读完之后除非我刻意地训练自己想起它，否则对我而言还是相当于没接触过——昨天的网球课上我收拍上肩的动作总是忘记做，我跟教练说能不能先恢复到不用收拍的动作练几下，教练说不行，你总忘记这个就要刻意提醒自己做到这个。

所以，没有输出的话大部分时候输入只是在已有的经验板上加深原本就有的痕迹，而我该拓展的经验地带则remain unchanged。长期下去我的输入只会把我推到我原本type的极端形态，除了初次接触时带来的思维上的惊叹，其他的理论和视角根本不会在我的世界留存下来。毛选实践论里说，拿到了理论，只是完成了一半任务，最重要的一半是用理论指导实践，在实践中检验理论。这样看的话我目前所做的事情主要还是停留在收集理论，甚至有些理论并没有被收集，只是从我手上滑过。

输出的最通用形态就是文字了，大部分东西都是语言可以承载和表达的。So，我会尽可能继续写下去。

另：

最近在线下沟通中时常感到词不达意，表达起来有点力不从心，总有些时刻寄希望于对方理解力够高，能够跟我同频，明白我blabla吐的一堆mess中串联起来想表达的逻辑是什么。但是实际上，我认为，大部分时间对方都是一头雾水地点点头，让我继续。

试图在日常沟通中做一个简单的人，此处简单意味着能够很容易让对方理解自己的意思。
